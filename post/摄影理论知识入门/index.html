常用镜头介绍

不同的品牌，原厂，副厂
  • 佳能：日本
  • 尼康：日本
  • 索尼：日本
  • 适马 ：日本，高质量，优惠，镜头性价比高
  • 腾龙：日本，，同上
不同的焦距：光线通过镜头后在承诺向平面上聚焦的位置。mm表示。决定视角和放大倍率。
  • 广角：焦距短，捕捉更宽广景象，适合风光，建筑
  • 中焦：接近人眼视角，更自然的景深和透视效果，更真实。适合人像，节拍，风景。
  • 长焦：捕捉较远处细节。景深小，虚化好。

其他分类
  • 变焦：焦距可调
  • 定焦：焦距固定，但性能好
  • 恒定光圈：光圈不随焦距改变。保持曝光一致。
  • 浮动光圈：光圈大小随焦距改变。经济。

相机曝光三要素：光圈，快门速度，感光度。

P8常用拍摄模式和适用题材

拍摄模式/曝光组合

定义：不同光圈快门的组合设定

作用：将一部分交给相机；减轻拍摄压力。

相机常用曝光组合及对应题材

  1. Auto-全自动：几乎用不到
  2. P-程序曝光/半自动：可以调ISO和EV；偷懒时可以使用
  3. A/AV-光圈优先：适合绝大多数场景；风光/人像/旅行。优先控制A，S自动，可调ISO&EV
  4. S/TV-快门优先：适合抓拍场景：体育，动物。优先控制S，A自动，可调ISO&EV
  5. M-手动曝光：误区：不只是只用M档才专业。A S ISO全部可手动控制，不可调节EV。特殊使用情况：
  • 亮度亘古不变的物体/固定参数（月亮）
  • 超复杂的光线（舞台）
  • 超出EV可调节的范围（>+-5EV）
  • 使用闪光灯
  1. B-B门/长时间曝光：风光慢门>30s长曝光使用。按住快门曝光，松开快门停止
  2. C-自定义：可设置自己常用参数

手机常用拍摄模式以及对应题材

  1. 原生相机：最常用拍摄模式。小技巧：多方式触发快门；连拍
  2. 夜景模式：常用；微光情况下拍摄；部分相机可手持几秒进行拍摄
  3. 慢门/长曝光/流光快门：拍摄风光（车流 云彩 水面 星轨 ）。大多数机型需使用三脚架
  4. 人像模式：人像拍摄使用；可以磨皮提供虚化效果
  5. HDR模式：应对大光比情况。保留曝光和阴影细节，获得更高动态范围。
  6. 全景模式：拍摄全景长图。适合风光以及创意拍摄
  7. 专业模式：使用较少，类似于相机M模式，只是不可调节光圈

P9对焦，景深，虚化

对焦

  1. 什么是对焦：半按快门，滴滴，对的是清晰的
  2. 景深的概念：焦平面，前后有一段清晰范围 。大光圈虚化强，前景深小景深。小光圈清晰，叫大景深深景深。也和焦距拍摄距离有关。

光圈>瞳孔径大小>焦深>景深。

  1. 对焦应该对在哪里

  2. 自动对焦和手动对焦：基本都用自动对焦。前景遮挡或太暗月全视食用手动对焦。

  3. 对焦模式：

  4. 焦点：拍哪里对哪里。眼部对焦。锁定对焦：半按快门hold住。

  5. 对焦哪里

  6. 为什么半按快门对不上焦：
  • 超过了最近的对焦距离；
  • 开启可手动对焦；
  • 对焦和快门分离了；af on后键对焦
  • AFON后键对焦的优势；
  • 场景太暗；
  • 场景没有反差

如何拍出好看的背景虚化

  1. 背景远离主体
  2. 当背景离主体近时候，相机靠近主体：微距。如手指放眼前，盯着手指看，背景是虚的。
  3. 用大光圈
  4. 焦距长：用长焦镜头

画面的虚实控制？？？

  1. 应该合焦：风光大场景。人物眼睛，物品。焦点合成，不同对焦。
  2. 可以虚化：长曝光轨迹，车轨，流云。杂乱的背景。
  3. 为什么画面拍虚了
  • 没对上焦：对在背景上
  • 景深太浅：
  • 快门速度过低：低于安全快门；低于运动速度。自己在高铁上。或拍运动的。
  • 手抖：手动，外界震动，用三脚架忘关防抖。
  • 建议：关键时刻可以连拍
